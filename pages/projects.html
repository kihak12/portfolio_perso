Voici les projets que j'ai pu réaliser lors de mes formations, où grâce à des projets personnel

  • Mise en service d'un serveur FTP distant

    Description du projet :

    C'est un projet personnel que j'ai réaliser, dans le but de pouvoir accéder depuis n'importe où aux données stocker sur mon ordinateur.

    Compétence travaillé et aquise :

    Utilisation du logiciel FileZilla Server, routage de ports, gestions d'accés et de partage des fichiers.

  • Programmation logiciel d'auto authentification

    Description du projet :

    C'est un projet personnel que j'ai réaliser, dans le but de pouvoir changer aisément et rapidement d'utilisateur sur une application.
    Concrètement l'utilisateur va entrer dans la basse de données du logiciel ces identifiants, puis lorsque l'utilisateur souhaitera se connecter au il n'aura qu'à se rendre sur la page de connexion du service, puis lancer le logiciel qui entrera lui même les identifiants.

    Language de programmation utiliser :

    AutoIt

  • Ajout de fonctionnaliter sur une app Android

    Description du projet :

    C'est un travail que j'ai réaliser lors de ma formation Prep-Android avec OpenClassrooms.
    Dans ce cas, j'ai eu pour mission d'ajouter un fonctionnaliter sur une app existante, cette fonctionnaliter consistait a afficher les informations d'un utilisateur dans une liste lors d'un clique sur celui-ci.
    J'ai également dû créer un bouton sur l'interface de l'utilisateur qui gérait l'ajout et la suppression de l'utilisateur dans une liste de favoris.
    Pour dernière mission j'eu à créer une nouvelle section sur la page d'acceuil de l'application qui liste les utilisateurs, afin de lister uniquement les utilisateurs marquées favoris.

    Language de programmation utiliser :

    AutoIt
